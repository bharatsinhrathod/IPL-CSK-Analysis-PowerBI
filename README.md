# IPL-CSK-Analysis-PowerBI
An interactive Power BI data analytics project exploring Chennai Super Kings (CSK) performance trends, match outcomes, and strategic success factors in the IPL (2008-2022).
# Strategic Analysis of Chennai Super Kings (IPL 2008-2022)

### Project Overview
This project provides a comprehensive data analysis of CSK's performance trends. 
It utilizes Power BI to transform raw match data into actionable insights.

### Key Features
* **Interactive Slicers:** Allows filtering match data by Season/Year.
* **Win Analysis:** Visualization of CSK victories vs. all IPL opponents.
* **Venue Insights:** Identification of high-performance stadiums (e.g., Chepauk).
* **Toss Strategy:** Analysis of win probability based on toss decisions.

### Technologies Used
* **Power BI Desktop:** For ETL (Power Query) and Visualization.
* **Excel/CSV:** Source data management.
* **DAX:** Custom measures for win-rate calculations.
